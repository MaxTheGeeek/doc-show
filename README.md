# 📄 DocShow

**DocShow** is a modern file-sharing platform that allows users to upload documents and share them via a link with **view-only access** — no downloads required.

---

## 🚀 Features

- 📤 Upload documents (PDF, images, Word files, etc.)
- 🔗 Share files via a secure link
- 👀 View documents directly in the browser (no download needed)
- 📱 Responsive and user-friendly interface
- 🔒 Controlled access to shared content

---

## 🧠 How It Works

1. Upload a file to the platform  
2. Generate a shareable link  
3. Open the link → the document is displayed instantly in the browser  
4. Users can **read or view** the file without downloading it  

---

## 🛠 Tech Stack

### Frontend
- Angular  
- TypeScript  
- Modern UI components  

### Backend
- C# / .NET  
- REST APIs  

### Cloud & Infrastructure
- AWS (deployment & services)  
- Cloud storage & database  

---

## 📦 Architecture

DocShow is built as a **full-stack application** with a clear separation between frontend and backend:

- Angular handles UI and document rendering  
- .NET backend manages file processing, APIs, and access  
- AWS is used for hosting, storage, and scalability  

---

## 🎯 Purpose

This project was built to:

- Demonstrate full-stack development skills  
- Showcase experience with **Angular + .NET + AWS**  
- Solve a real-world problem: **secure document sharing without download access**
